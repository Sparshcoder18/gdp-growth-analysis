# Cross-Country Analysis of Economic Growth Trends Using GDP (Current US$) Data (1970–2024)

## 📌 Project Overview
This project performs a descriptive cross-country analysis of GDP (Current US$) from 1970 to 2024. The study focuses on long-term economic growth patterns, annual growth rates, decade-wise fluctuations, and comparative economic progression among nations.

## 🎯 Objectives
- Analyze long-term GDP trends (1970–2024)
- Compute annual growth rates
- Calculate average growth rates
- Compare GDP performance across countries
- Analyze decade-wise growth fluctuations
- Identify relative economic progression

## 📊 Dataset
- Source: United Nations Statistics Division
- Indicator: GDP (Current US$)
- Time Range: 1970–2024

## 🛠 Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

## 📈 Key Analysis Performed
- Data Cleaning & Preprocessing
- Year-over-Year Growth Rate Calculation
- Decade-based Aggregation
- Cross-country Comparative Visualization
- Descriptive Statistical Analysis
